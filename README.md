# FinTrust Capital Website

A responsive finance website for **FinTrust Capital**, built entirely with **pure HTML** and **Tailwind CSS**.  
This project showcases a professional design with clear navigation and dedicated pages for services, insights, and client engagement.

---

## Project Structure

The site consists of the following pages:

- **Home (`index.html`)**  
  Landing page with hero section, company overview, and navigation.

- **Services**
  Details of financial solutions offered, including investment advisory, risk management, and SME financing.

- **Insights**  
  Blog-style page with articles, market updates, and thought leadership content.

- **About Us**  
  Company vision, mission, values, and testimonials.

- **Contact**  
  Contact form, office address, email, phone, and embedded Google Map.

- **Login**  
  Secure login portal with email, password, and “Remember Me” option.

- **Sign Up**  
  Registration form with name, email, password, confirm password, and terms acceptance.

---

## Technologies Used

- **HTML** – Semantic markup for structure.
- **Tailwind CSS** – Utility-first CSS framework for styling and responsiveness.

## Features

- **Responsive Navigation Bar**  
  Fixed header with dropdown menu for mobile.

- **Finance Branding**  
  Blue/black palette for trust and professionalism.

- **Forms**
  - Contact form with name, email, subject, priority, and message.
  - Sign Up form with validation fields.
  - Login portal with “Remember Me” and “Forgot Password” links.

- **Testimonials Section**  
  Client feedback highlighting trust and satisfaction.

- **Insights Blog**  
  Articles and updates to build credibility and thought leadership.
